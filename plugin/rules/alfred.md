# alfred MCP Tools

alfred is your silent butler for Claude Code. He never interrupts — only helps when called.

Use alfred's MCP tools when the following conditions apply.

## knowledge — Search Claude Code docs and best practices

USE when:
- Looking up Claude Code features (hooks, skills, rules, agents, MCP, memory, CLAUDE.md)
- Creating, modifying, or reviewing `.claude/` configuration files (check best practices first)
- Verifying correct format or structure for configuration files

## review — Analyze project's Claude Code utilization

USE when:
- Reviewing or auditing `.claude/` configuration (agents, skills, rules, hooks, MCP)
- Evaluating CLAUDE.md quality or looking for improvements
- Checking overall Claude Code setup health for a project

## ingest — Store documentation in knowledge base

USE when:
- Ingesting crawled documentation into the knowledge base
